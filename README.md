# # BookBot

BookBot is a Python command-line application that analyzes books and text files, generating reports that include total word count and character frequency rankings.

## Technologies Used

* Python 3
* File I/O
* Dictionaries
* Custom sorting algorithms
* Type hints
* Multi-module project structure
* Git & GitHub

## How to Run

```bash
Usage:
python3 main.py <path_to_book>

Example:
python3 main.py books/frankenstein.txt
```

## Example Output

```text
============ BOOKBOT ============
Analyzing book found at books/frankenstein.txt...
----------- Word Count ----------
Found 75767 total words
--------- Character Count -------
e: 44538
t: 29493
a: 25894
...
============= END ===============
```

## What This Project Demonstrates

* Text file reading and processing using Python file I/O
* Dictionary-based data aggregation and frequency analysis
* Custom sorting with key functions
* Type hints for cleaner, more maintainable code
* Multi-file module structure with separation of concerns
* Command-line application development
* Version control using Git and GitHub
